# Releases এ EXE দেখানো ও ডাউনলোড

আপনি (অথবা যেকোনো ব্যবহারকারী) GitHub দিয়ে বানানো EXE টা **Releases** থেকে এভাবে ডাউনলোড করতে পারবেন:

1. **রিপো পেজে যান:**  
   [https://github.com/adnanahamed66772ndpc/MALER](https://github.com/adnanahamed66772ndpc/MALER)

2. **Releases খুলুন:**  
   পেজের ডান পাশে **"Releases"** লিংক এ ক্লিক করুন (অথবা সরাসরি: [https://github.com/adnanahamed66772ndpc/MALER/releases](https://github.com/adnanahamed66772ndpc/MALER/releases))।

3. **যে ভার্সন চান সেটা বেছে নিন:**  
   নিচের দিকে যে ভার্সন গুলো আছে (যেমন **v1.0.0**) তার ওপর ক্লিক করুন।

4. **ZIP ডাউনলোড করুন:**  
   সেই রিলিজের নিচে **Assets** সেকশনে ফাইল দেখা যাবে:
   - **MailerApp-Desktop-win-x64.zip** – ডেস্কটপ অ্যাপ (এটাই মূল EXE সহ অ্যাপ)। এইটা ডাউনলোড করুন।
   - MailerApp-Worker-win-x64.zip – Worker (ইমেল সেন্ড করার জন্য, প্রয়োজন হলে ডাউনলোড করুন)।

5. **ZIP আনজিপ করে চালান:**  
   ডাউনলোড করা ZIP আনজিপ করুন। ভেতরে **MailerApp.Desktop.exe** থাকবে – সেটা ডাবল-ক্লিক করেই অ্যাপ চালাতে পারবেন।

## প্রতিটি পুশে নতুন রিলিজ (অটো)

**main** ব্রাঞ্চে যেকোনো push করলেই অটো একটা নতুন রিলিজ তৈরি হবে এবং তাতে নতুন EXE জিপ আপলোড হবে।  
রিলিজের নাম হবে **Build 1**, **Build 2**, … (প্রতি পুশে নম্বর বাড়বে)।  
কোনো আলাদা ট্যাগ বা কমান্ড দরকার নেই – শুধু `git push origin main`।

## ভার্সন রিলিজ (ট্যাগ পুশ)

GitHub এর **Releases** পেজে EXE (ZIP) দেখাতে চাইলে একটা **ট্যাগ** পুশ করলেই হবে। ট্যাগ পুশ করলে অটো বিল্ড হবে এবং সেই ভার্সনের রিলিজে EXE জিপ আপলোড হবে।

## ধাপ ১: কোড পুশ করুন

প্রথমে সব কোড main এ পুশ করুন (যেমনটা আগে করছেন):

```bash
cd C:\Users\Adnan\Documents\MAILER
git add -A
git commit -m "Update"
git push origin main
```

## ধাপ ২: ভার্সন ট্যাগ দিন ও পুশ করুন

যে ভার্সন দিয়ে রিলিজ চান (যেমন 1.0.0), সেই অনুযায়ী ট্যাগ বানিয়ে পুশ করুন:

```bash
git tag v1.0.0
git push origin v1.0.0
```

- `v1.0.0` এর জায়গায় যেকোনো ভার্সন দিতে পারেন: `v1.0.1`, `v2.0.0` ইত্যাদি।
- **v** দিয়ে শুরু করতে হবে (যেমন v1.0.0)।

## ধাপ ৩: রিলিজ দেখুন

- ট্যাগ পুশ করলে GitHub Actions অটো বিল্ড চালাবে।
- বিল্ড শেষ হলে **Releases** পেজে নতুন রিলিজ চলে আসবে (ট্যাগের নামে)।
- সেখানে **MailerApp-Desktop-win-x64.zip** ও **MailerApp-Worker-win-x64.zip** ডাউনলোড লিংক থাকবে।

রিপো পেজে গিয়ে **Releases** (ডান পাশে) ক্লিক করলেই সব রিলিজ ও EXE জিপ দেখা যাবে।

## সংক্ষেপে

| করণীয়              | ফলাফল                    |
|----------------------|---------------------------|
| `git tag v1.0.0`     | লোকালে v1.0.0 ট্যাগ তৈরি |
| `git push origin v1.0.0` | ট্যাগ পুশ → Actions বিল্ড → Releases এ EXE জিপ |

পরবর্তী ভার্সনের জন্য নতুন ট্যাগ দিন (যেমন `v1.0.1`) এবং আবার পুশ করুন।
